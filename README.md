## Set Up Instructions

* Make sure Ruby, Rails, and PostgreSQL are installed.
* Clone Git Repository: https://github.com/SiR-Frazier/Wk3RailsReview
* cd wk3RailsReview
* run bundle install
* run rails db:create
* run rails db: migrate
* run rails db:seed
* run rails s
* go to localhost:3000 in your preferred browser.
